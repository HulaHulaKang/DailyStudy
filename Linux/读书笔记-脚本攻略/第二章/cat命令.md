### cat命令:显示文件内容
单词解释：cat ———— concatenate(拼接)


```shell
同时读取多个文件：
cat file1 file2 file3 ...
#可用 此功能+重定向 实现合并文件的结果
```

```shell
将一个字符串和文件内容拼接显示:

echo '...' | cat - filename
# 排列顺序为：字符串内容 、filename 的内容
# 短横线是字符串的占位符

echo '...' | cat file1 - file2 # -表示前面的字符串
# 排列顺序为：file1的内容 、字符串内容 、file2
# 短横线是字符串的占位符
```

```shell
-s选项  #当遇到有连续两行以上的空白行，就代换为一行的空白行。

-T选项  #将制表符显示为^I(以便与空格区分开)

-n选项  #显示行号
```
