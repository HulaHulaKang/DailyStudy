# JavaScript中的for语句  

## 用法一：同C++

```PlainText
注意第一个参数用let声明
```

## 用法二：for-in

```JS
//for...in 语句循环一个数组（Array）的下标（键），注意，是下标
for (variable in object) {
  statements
}
//variable是数组的下标
```

## 用法三：for-of

```JS
//for...in 语句循环一个数组（Array）的元素的值，注意，是值
for (variable in object) {
  statements
}
//variable是数组的元素的值
```
